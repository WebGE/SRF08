Télémètre à ultrasons "SRF08"

  • Présentation

    • Ce module est conçu pour les applications de télémétrie pour robotique ludique. Il est capable de déterminer la distance qui le sépare d'un obstacle se présentant devant lui (entre 3 cm et 6 m). Doté de 2 cellules à ultrasons, son principe de fonctionnement repose sur celui des "sonars". Il s'interface à l'aide de son bus I2C™ et se pilote à la manière d'une mémoire EEPROM type 24xx . Il peut retourner la valeur de la distance en "mm", en "inch" ou sous la forme d'une durée (en µs) liée à l’écho de l'émetteur ultrason.

  • Diagramme de rayonnement

  • Alimentation : 5V (15mA Typ. 3mA Standby).
  • Possibilités : Il est possible d'adresser jusqu'à 16 modules différents par le bus I2C™.
  • Dimensions : 43mm w x 20mm d x 17mm h
  • Documentation : Devantech SRF08 UltraSonic Ranger
  • Distributeur : Lextronic

Classe SRF08

Contenu du dossier à télécharger

Le dossier à télécharger (au format zip ou tar.gz) contient le répertoire d'un projet Microsoft Visual Studio Professionnel composé :

  • Du code de la classe SRF08.
  • D'un exemple d'utilisation de cette classe.

Les fichiers program.cs et DriversSRF08.cs pourront être recompilés avec Microsoft Visual Studio Express.

Utilisation

  1. Créer un objet en utilisant un des constructeurs SRF08.
  2. Utiliser la méthode ReadRange pour déclencher et lire la distance.

Constructeurs

Syntaxe Description
I2CLcd() Instancie un objet "I2CLcd" avec les paramètres :
Fabricant = Midas
Fréquence du bus: 100kHz
I2CLcd(UInt16 I2C_Add_7bits) Instancie un objet "I2CLcd".
I2C_Add_7bits est l'adresse I2C sur 7 bits attribuée à l'afficheur.
Fréquence du bus: 100kHz.
I2CLcd(LcdManufacturer ManufacturerName) Instancie un objet "I2CLcd". ManufacturerName est le nom du fabricant.
Il correspond à l'adresse I2C du LCD :
  • Midas (0x3a)
  • Batron (0x3b)
Fréquence du bus: 100kHz.
I2CLcd(UInt16 I2C_Add_7bits, UInt16 Frequency) Instancie un objet "I2CLcd".
I2C_Add_7bits est l'adresse I2C sur 7 bits attribuée à l'afficheur.
Frequency est à définir dans l'intervalle [100kHz,400kHz].
I2CLcd(LcdManufacturer ManufacturerName, UInt16 Frequency) Instancie un objet "I2CLcd". ManufacturerName est le nom du fabricant.
Il correspond à l'adresse I2C du LCD :
  • Midas (0x3a)
  • Batron (0x3b)
Frequency est à définir dans l'intervalle [100kHz,400kHz].

Enumérations
